# School System

This is a personal project for a school management system built with PHP.

## Features

- User authentication (login/logout)
- Role-based access for admin, cashier, registrar, teacher, and user management
- Responsive design with CSS

## Installation

1. Clone the repository: `git clone https://github.com/HitzoChan/school_system.git`
2. Set up XAMPP or a similar PHP server.
3. Place the project in `htdocs` directory.
4. Configure database in `config.php`.
5. Run the application via localhost.

## Usage

Access the application through your web browser at `http://localhost/school_system`.
